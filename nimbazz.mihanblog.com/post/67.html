KEmulator V.1.0.0 شبیه ساز جاوا برای Pc

نرم افزار KEMULATOR بهترین نرم افزار برای اجرای فایلهای بازی و برنامه موبایل جاوا با پسوند JAR هست. پیش از این چندین نرم افزار مانند SJBOY و FREEWARE MIDPX فایل جاوا را اجراء می کردند. اما هر یک نواقص زیادی داشت که در این نرم افزار برطرف شده است.

قابلیت های نرم افزار:
- امکان اجرای اتوماتیک فایل های JAR و JAD
- تنظیم با همه سایزها و رزولیشن ها حتی سایزهایی مانند 1366*768
- ضبط تصاویر بصورت فیلم و عکس از بازی ها و برنامه ها
- اتصال به اینترنت کامپیوتر بدون دنگ و فنگ
- تنظیمات حرفه ای فونت های نمایشی
- کیبورد مجازی
تنها ایراد این برنامه نیاز به JAVA RUNTIME برای اجرای آن است که می توانید از زیر دانلود کنید.

دانلود شبیه ساز جاوا KEmulator V.1.0.0 با حجم 3.25mb

دانلود Java SE Runtime Environment 7.0.25 نسخه 32 بیتی با لینک مستقیم(28 MB)

دانلود Java SE Runtime Environment 7.0.25 نسخه 64 بیتی با لینک مستقیم(29.52 MB)
